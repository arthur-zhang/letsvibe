# Tauri + React + TypeScript

This project is a Tauri desktop application built with React 18 and TypeScript, using Vite as the build tool. The application was migrated from Vue 3 to React 18, maintaining the same functionality while leveraging React's component model and hooks.

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Desktop Framework**: Tauri 2.x
- **Build Tool**: Vite
- **Styling**: TailwindCSS
- **State Management**: React hooks and context
